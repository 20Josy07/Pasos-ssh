1. Subir archivo (SCP)

Desde tu PC (usa tu NUEVA clave personal):

scp dbparasql.sql estuserdb1@formxa.com:/home/estuserdb1/

2. Importar a MySQL

Dentro del servidor (aquí la clave sigue siendo la inicial):

mysql -h 127.0.0.1 -P 3307 -u estuserdb1 -p db1 < dbparasql.sql

DB Name: db1 | Clave MySQL: pass1

Entra al monitor de MySQL:

mysql -h 127.0.0.1 -P 3307 -u estuserdb1 -p
